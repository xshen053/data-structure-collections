# Map Interface

`Map61B.java` is an interface, it includes all the operations the Map ADT has
- `void put(K key, V value)`
- `V get(K key)`
- `int size()`
- `void clear()`
- `V remove(K key)`


# 3 implementations of map

- Binary Search Tree
- Red Black Tree
- Unordered Linked List
